
# Nedbank Money (Firebase Studio)

This is the Nedbank Digital Banking platform for Trusts, featuring both a Web and Mobile experience.

## 🌐 Web App
To get started with the web version, the app is automatically running in the preview window.
- **Main Dashboard**: Accessible for standard clients.
- **Trustee Portal**: Accessible for users with the `trustee` role.

## 📱 Mobile App
The mobile version is built with React Native and Expo.
- For instructions on how to run it on your phone or download the APK, see: [mobile/README.md](./mobile/README.md)

### Quick Run:
```bash
npm run mobile
```
