# **App Name**: MoneyGO

## Core Features:

- Account Overview: Display a summary of all linked Nedbank accounts and balances.
- Transaction History: View a detailed history of transactions for each account.
- Bill Payments: Make payments to existing beneficiaries. Select existing beneficiaries, and automate common payments.
- Funds Transfer: Transfer funds between Nedbank accounts and to external recipients.
- ATM Locator: Find nearby Nedbank ATMs using the device's location.
- Personalized Financial Tips: Provide AI-powered personalized tips on how to manage money effectively. The LLM will use your income, spending habits, and the 'Nedbank Budget Tool' to determine how useful it should make a tool offering tips to the user.
- Biometric Authentication: Secure access to the app using fingerprint or facial recognition.
- Implement full Features of Nedbank: Must be a full blueprint of nedbank money
- Share Proof of Payment: Be able to share proof of payment Pdf documents

## Style Guidelines:

- Primary color: Deep blue (#1A498B), evoking trust and stability.
- Background color: Light blue (#E6F0FA), providing a clean and modern feel.
- Accent color: Teal (#26A69A), highlighting key actions and elements.
- Headline font: 'Poppins', a geometric sans-serif, for a contemporary and precise style.
- Body font: 'PT Sans', a humanist sans-serif, balancing modernity and readability.
- Use clean, modern icons representing financial transactions and features.
- Maintain a simple, intuitive layout for easy navigation and accessibility.
