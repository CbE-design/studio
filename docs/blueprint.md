# **App Name**: Van Schalkwyk Trust Mobile

## Core Features:

- Account Overview: Display live account balances for the Van Schalkwyk Family Trust.
- Transaction History: Show a detailed view of all transactions, sorted by date.
- Payment Initiation: Enable users to enter recipient details and payment amount. Tool: Validate banking details and provide transaction summaries.
- Payment Confirmation: Display a confirmation screen with transaction details after a payment is made.
- Personalization: Enable trust managers to specify custom transaction categories.

## Style Guidelines:

- Primary color: Deep sky blue (#3399FF) for trust and clarity.
- Background color: Light gray (#F0F0F0) to ensure legibility.
- Accent color: Teal (#008080) for interactive elements.
- Body and headline font: 'Inter', a sans-serif, for a clean and modern user experience.
- Use clear, minimalist icons for navigation and actions.
- Emphasize clarity and readability with a card-based design.
- Use subtle animations to indicate changes and interactions.
